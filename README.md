# Java Core — Practice Programs

A collection of Java programs covering core language fundamentals — conditionals, loops, sorting logic, and basic input/output.

## Topics Covered

- Conditional statements (`if-else`, nested conditions)
- Sorting logic without built-in methods
- Number manipulation and mathematical operations
- Iterative patterns using loops

## Files

| File | Description |
|---|---|
| `A.java` | Core Java fundamentals exercise |
| `Test.java` | Basic conditional and output logic |
| `Test1.java` – `Test17.java` | Progressive exercises covering sorting, number comparisons, and loop patterns |
